full-Stack Personal Portfolio Hub
A professional, full-stack application built to showcase technical projects and development skills. This platform demonstrates a clean integration between a React frontend and a Node.js/Express backend, featuring real-time data management and a modern, responsive user interface.

Key Features
Dynamic Project Management: Full CRUD capability allowing users to add and delete projects instantly.

Live Search & Filtering: Real-time JavaScript-based search engine to filter the project list by title.

Project Categorization: Automated tagging system (Web Dev, Blockchain, Mobile) to organize technical work.

Responsive Dashboard: A mobile-friendly interface designed with modern "Glassmorphism" UI principles.

Asynchronous API Operations: Robust communication between client and server using Axios and Async/Await patterns.

Tech Stack
Frontend
React.js: Component-based architecture for a reactive user experience.

CSS3: Custom styling utilizing CSS variables and advanced layout techniques (Flexbox/Grid).

Axios: Handling all asynchronous HTTP requests to the backend API.

Backend
Node.js: Scalable JavaScript runtime for server-side logic.

Express.js: Lightweight framework for building RESTful API routes.

CORS: Middleware for secure Cross-Origin Resource Sharing.
